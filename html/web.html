The Internet - The World Wide Web

Web Pages

A web page is an HTML document located on the internet using a URL that consists of a transfer protocol, domain name or IP address, and file path. Browsers can store website addresses as bookmarks or favorites. To obtain a URL, the browser sends the domain name to a DNS server to determine the IP address of the host computer that will supply the page, then builds a request for the page and transmits it onto the internet. The server processes the request, returning either an existing page or building a new one.

Hyperlinks

Example of a Hyperlink

Ted Nelson came up with the idea of words with attached explanations in the 1960s. Tim Berners-Lee later imagined hyperlinked documents, where important words act like buttons and take the reader on a journey where they can read more. HTML usually presents these links as blue underlined words, images, or icons, and clicking on them launches a request for the linked item, which is then delivered for viewing.
