The Internet - Software

Browsers

The two main internet software applications are browsers and email clients. Browsers interpret HTML coding to display text, graphics, and media, and have become more sophisticated over time, allowing for videoconferencing and email functions. Operating systems are trending towards full online integration with the web, but require a fast internet connection. HTML uses paired tags to describe page content and can be created using text editors or special software. Browsers can also use scripting languages and plug-ins for additional features such as reading PDFs, playing media, and viewing animations. Further details on browsers and the web are discussed later in the chapter.

Style Sheet Languages

In the past, HTML tags were used to describe the appearance of every part of a web page. However, CSS now takes care of these details by defining styles for different parts of a page. CSS allows web designers to attach a style sheet that lists font size, color, and text alignment. This information is kept in a special area in the document and can be replaced by a user's own style sheet. Visually impaired users can take advantage of CSS by enlarging font sizes and using special colors.

Markup Languages

Markup languages use tags to annotate text with formatting and structure information, defining the hierarchy of headings, subheadings, and sections. HTML defines the structure and content of web pages, while CSS defines the style and layout. XML is used for data exchange and storage, enabling the creation of custom tags and attributes. Markup languages are essential for creating structured, visually appealing documents and web pages that are readable and easy to understand, while also allowing for customization.

Plug-in Technologies

A plug-in refers to an application that can be incorporated into a browser or added to it, which allows the browser to interact with a particular file type. The Flash plug-in, for instance, facilitates the viewing of Flash animations in the browser through SWF files (pronounced 'swiff').

Client-side Scripting

Scripts that run on the user's computer are referred to as client-side scripts. JavaScripts, for example, can enhance HTML pages by providing additional functionality, such as modifying an image when the user hovers over it as an indication to click on the image. Java applets, which differ from JavaScript, are another type of client-side scripting that can create advanced interactions with web pages.

Server-side Scripting

Scripts that run on the server hosting web pages are known as server-side scripts. To enhance the interactivity of web pages, other languages such as PHP have been developed to enable dynamic web page features. PHP, which stands for Personal Home Page, is widely used today. Many popular websites such as Facebook, Wikipedia, and Moodle use PHP scripts to present their pages to users.

Database Technologies

Frequently, it becomes necessary to access data stored in databases. In such cases, the SQL (pronounced 'see-kwel') language provides a solution by allowing database access, thereby serving as another form of server-side technology.

HTML5

HTML5 is a technology that reduces the need for third-party plug-ins, such as Flash, to handle multimedia and graphics. It is designed to be mobile-friendly and has features such as video and audio support, vector graphics, drag and drop, and mathematical formulas.

Website Design Tools vs Authoring Software

Website design tools like IM Creator and SquareSpace allow users to create a website without coding, limited to the built-in tools. Professional website authoring software like Dreamweaver offer more control and require mastery of web-based coding languages. Advanced authoring applications can help manage the whole site and check for inconsistencies in links.
